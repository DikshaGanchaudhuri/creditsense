# CreditSense — AI-Augmented Credit Risk Analytics Dashboard

A full-lifecycle data science project: from raw data ingestion to ML modeling,
interactive Tableau dashboards, and a LangChain + Gemini AI narration layer.

**Stack:** Python · PostgreSQL · Tableau · LangChain · Gemini API

## Project Structure
- `data/` — raw and processed datasets
- `notebooks/` — exploratory and modeling notebooks
- `src/` — reusable Python modules
- `sql/` — database schema and queries
- `dashboard/` — Tableau workbook
- `ai_layer/` — LangChain + Gemini pipeline
